# Motor Text to Speech (TTS) en Rust

Motor de conversión de texto a audio desarrollado en Rust. El sistema genera
archivos de audio en formato MP3 de manera local, sin requerir conexión a internet,
priorizando rendimiento y eficiencia.

## Funcionalidades
- Conversión de texto a voz
- Generación de archivos MP3
- Ejecución local (offline)
- Procesamiento eficiente de texto

## Tecnologías utilizadas
- Lenguaje: Rust
- Generación de audio: MP3
- Ejecución local sin dependencias externas

## Rol personal
Diseño e implementación completa del motor TTS, enfocado en rendimiento y uso local.
